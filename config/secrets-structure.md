# Структура хранилища секретов

## Обзор

Этот документ описывает детальную структуру хранилища секретов. Используйте его как справочник при настройке хранилища секретов (1Password, Bitwarden, Notion и т.д.).

## Рекомендации по организации

### В 1Password / Bitwarden

Создайте отдельную папку "Dropshipping System" и внутри неё создайте отдельные записи для каждого сервиса.

### В Notion

Создайте закрытую страницу "Secrets" с ограниченным доступом. Используйте таблицу или отдельные страницы для каждого сервиса.
