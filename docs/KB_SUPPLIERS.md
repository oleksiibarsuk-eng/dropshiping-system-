# База Suppliers

## Назначение

Систематизирует поставщиков и источники закупок. В отличие от Products Research, фокус здесь на контрагентах, а не на конкретных SKU. Это позволяет видеть карту надежных и рискованных поставщиков и принимать более взвешенные решения о том, кому доверять высокомаржинальные или сложные в логистике позиции.

## Структура базы данных

### Основные поля

#### Supplier name
- **Тип**: Текст
- **Описание**: Наименование поставщика (магазин, eBay‑продавец, платформа)
- **Пример**: "Camera Store Berlin", "eBay Seller: photo_equipment_de"

#### Type
- **Тип**: Select
- **Описание**: Тип поставщика
- **Варианты**: 
  - магазин
  - частный продавец
  - платформа
  - оптовик
- **Пример**: "магазин"

#### Country/Region
- **Тип**: Select
- **Описание**: Страна/регион (важно для логистики, таможни и скорости доставки)
- **Варианты**: Германия, Япония, США, другие
- **Пример**: "Германия"

#### Link / Contacts
- **Тип**: URL + Текст
- **Описание**: Ссылки на профили, сайты, контактные e‑mails, телефоны
- **Пример**: 
  - Website: https://camerastore.de
  - Email: info@camerastore.de
  - eBay Profile: https://www.ebay.de/usr/...

#### Products
- **Тип**: Relation (к Products Research) или Multi-select
- **Описание**: Связь с Products Research или набор тегов категорий, по которым поставщик интересен
- **Пример**: ["Sony A7III", "Canon EF Lenses", "Fuji Cameras"]

#### Average handling time
- **Тип**: Number
- **Описание**: Среднее время обработки заказов (по наблюдениям и/или заявленным SLA)
- **Единица измерения**: дни
- **Пример**: 2

#### Shipping methods & стоимость
- **Тип**: Текст (длинный)
- **Описание**: Доступные способы доставки и ориентировочные стоимости по ключевым направлениям
- **Пример**: 
  - DHL Standard: €15-25 (5-7 дней)
  - DHL Express: €35-50 (2-3 дня)
  - Доставка в США: €30-45 (10-14 дней)

#### Return policy
- **Тип**: Текст
- **Описание**: Политика возвратов: сроки, условия, кто оплачивает обратную доставку
- **Пример**: "14 дней на возврат, покупатель оплачивает доставку, товар должен быть в оригинальной упаковке"

#### Reliability score (1–5) + комментарий
- **Тип**: Number + Текст
- **Описание**: Субъективный рейтинг надёжности на основе опыта
- **Критерии оценки**:
  - 5: Идеально, никогда не было проблем
  - 4: Хорошо, редкие мелкие проблемы
  - 3: Удовлетворительно, периодические проблемы
  - 2: Плохо, частые проблемы
  - 1: Очень плохо, постоянные проблемы
- **Пример**: 4, "Один раз была задержка на 2 дня, но в целом надёжный"

#### Notes
- **Тип**: Текст (длинный)
- **Описание**: Зафиксированные кейсы (успешные и проблемные), особенности коммуникации, договорённости
- **Пример**: 
  - "Всегда быстро отвечает на вопросы"
  - "Один раз отправил товар с небольшим дефектом, но быстро заменил"
  - "Специальная скидка 5% при заказе от 3 товаров"

### Дополнительные поля

#### Last order date
- **Тип**: Date
- **Описание**: Дата последнего заказа у этого поставщика
- **Пример**: 2024-01-15

#### Total orders
- **Тип**: Number
- **Описание**: Общее количество заказов
- **Пример**: 12

#### Success rate
- **Тип**: Number (процент)
- **Описание**: Процент успешных заказов
- **Пример**: 95%

#### Status
- **Тип**: Select
- **Описание**: Текущий статус работы с поставщиком
- **Варианты**: 
  - active — активно используем
  - testing — тестируем
  - paused — временно приостановлено
  - blocked — заблокирован
- **Пример**: "active"

## Примеры записей

### Пример 1: Надёжный магазин

```
Supplier name: Camera Store Berlin
Type: магазин
Country/Region: Германия
Link: https://camerastore.de, info@camerastore.de
Products: ["Sony Cameras", "Canon Lenses", "Fuji Accessories"]
Average handling time: 1 день
Shipping methods: DHL Standard €20 (5-7 дней), Express €40 (2-3 дня)
Return policy: 14 дней, покупатель оплачивает доставку
Reliability score: 5
Notes: Очень надёжный, быстрая обработка, качественная упаковка
Last order date: 2024-01-20
Total orders: 25
Success rate: 100%
Status: active
```

### Пример 2: eBay продавец

```
Supplier name: photo_equipment_de (eBay)
Type: частный продавец
Country/Region: Германия
Link: https://www.ebay.de/usr/photo_equipment_de
Products: ["Vintage Cameras", "Collectibles"]
Average handling time: 3 дня
Shipping methods: DHL Standard €15 (7-10 дней)
Return policy: 30 дней, продавец оплачивает доставку
Reliability score: 3
Notes: Иногда задержки, но товары всегда соответствуют описанию
Last order date: 2024-01-10
Total orders: 5
Success rate: 80%
Status: testing
```

## Рекомендации по использованию

1. **Регулярное обновление**: Обновляйте Reliability score и Notes после каждого заказа
2. **Документирование проблем**: Фиксируйте все проблемы для анализа паттернов
3. **Сравнение**: Используйте базу для сравнения поставщиков по ключевым метрикам
4. **Связь с Products**: Связывайте поставщиков с товарами для быстрого поиска альтернатив

## Интеграция с системой

База Suppliers интегрируется с:
- **Products Research** — связь через поле Products
- **Supabase** — таблица suppliers синхронизируется с этой базой
- **Multi-Sourcing Agent** — использует базу для выбора оптимального поставщика
- **n8n workflows** — для автоматического обновления метрик

## Стратегические решения

Со временем именно эта база становится источником стратегических решений:
- Кого оставить ключевым партнёром
- С кем ограничить объём работ
- С кем, возможно, полностью прекратить сотрудничество
