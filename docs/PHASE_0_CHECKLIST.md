# Фаза 0: Ручная Настройка Инфраструктуры

## Важное понимание

Эти шаги невозможно автоматизировать из-за требований верификации личности, KYC процедур и привязки банковских карт. Вы выполняете их один раз руками, после чего агенты берут управление операциями на себя.

Общее время выполнения: от четырех до семи дней с учетом времени верификации аккаунтов.

### Shopify E-commerce Platform
**Время выполнения**: тридцать минут  
**Стоимость**: тридцать девять долларов в месяц или пробный период четырнадцать дней

**Детальный гайд**: [SETUP_SHOPIFY.md](SETUP_SHOPIFY.md)

**Шаги выполнения:**

Зарегистрируйтесь на платформе shopify.com создав новый аккаунт. Выберите подходящий план где Basic план за тридцать девять долларов в месяц содержит все необходимые функции для старта, либо активируйте бесплатный пробный период на четырнадцать дней для тестирования системы.

Заполните детали магазина включая название которое будет частью URL вашего магазина, физический адрес для юридической информации и валюту в которой будут указаны цены, рекомендуется USD для американского рынка или EUR для европейского.

Создайте Admin API Access Token перейдя в Settings затем Apps and sales channels, далее Develop apps и Create app. Настройте Admin API scope выбрав необходимые права: read_products для чтения информации о товарах, write_products для создания и изменения товаров, read_orders для просмотра заказов, write_orders для обновления статусов заказов.

Сохраните полученные API Key и Admin Access Token в безопасное место так как они потребуются для интеграции с n8n workflows.

### eBay Marketplace Accounts
**Время выполнения**: от двух до трех часов плюс ожидание верификации до сорока восьми часов  
**Стоимость**: бесплатная регистрация плюс комиссии с продаж около тринадцати процентов

**Детальный гайд**: [SETUP_EBAY.md](SETUP_EBAY.md)

**Шаги выполнения:**

Зарегистрируйте аккаунт на eBay.com для продаж на американском рынке который обеспечивает более высокие цены и больший объем покупателей. Затем зарегистрируйте отдельный аккаунт на eBay.de для закупок товаров по более низким ценам благодаря арбитражу.

Выполните апгрейд до Business Account на обоих аккаунтах что дает доступ к расширенным инструментам продавца и снижает некоторые ограничения для новых аккаунтов.

Пройдите процесс верификации загрузив скан паспорта или ID документа для подтверждения личности. Привяжите банковскую карту для оплаты комиссий платформы. Подтвердите физический адрес через письмо с кодом или загрузку документа подтверждающего адрес регистрации.
Создайте Developer Account перейдя на developer.ebay.com в раздел My Account и выбрав Create Keyset. Этот шаг критически важен для программного доступа к eBay через API.

Получите Production tokens которые включают Application ID также называемый Client ID используемый для идентификации вашего приложения, Developer ID привязанный к вашему аккаунту разработчика, Certificate ID или Client Secret для безопасной аутентификации, и User Token полученный через OAuth процесс для доступа к вашему продающему аккаунту.

Настройте Business Policies что упростит создание листингов в будущем. Создайте Payment Policy установив требование немедленной оплаты Immediate payment required чтобы избежать неоплаченных заказов. Создайте Shipping Policy с вариантами Standard delivery занимающей от семи до пятнадцати дней как экономичный вариант и Expedited delivery от трех до семи дней как более быстрая доставка за дополнительную плату. Настройте Returns Policy предлагая тридцать дней на возврат с гарантией возврата денег money back что повышает доверие покупателей и улучшает ранжирование листингов.

Сохраните все полученные токены в безопасном месте так как они потребуются для настройки n8n интеграций.

### Meta Business Suite для Facebook и Instagram
**Время выполнения**: от одного до двух часов плюс ожидание верификации от трех до пяти дней  
**Стоимость**: бесплатная регистрация плюс расходы на рекламу по вашему выбору

**Детальный гайд**: [SETUP_META.md](SETUP_META.md)

**Шаги выполнения:**

Создайте Business Manager перейдя на business.facebook.com и заполнив информацию о вашем бизнесе. Это центральная платформа для управления всеми бизнес-активами на Facebook и Instagram.

Добавьте Facebook Page создав новую страницу для вашего бизнеса или подключив существующую страницу к Business Manager. Страница будет использоваться для публикации контента и запуска рекламы.

Подключите Instagram Business Account который позволит продавать товары через Instagram Shopping и запускать рекламные кампании в Instagram Feed и Stories.

Заполните полную бизнес информацию включая Legal business name официальное юридическое название вашей компании или ваше полное имя если работаете как индивидуальный предприниматель, Business address физический адрес регистрации бизнеса и Website URL вашего магазина Shopify для подтверждения легитимности.

Подайте заявку на Business Verification что критически важно для доступа к расширенным возможностям рекламы. Загрузите документы компании такие как выписка из реестра юридических лиц или DBA certificate если работаете в США. Предоставьте подтверждение адреса через банковскую выписку или счет за коммунальные услуги. Процесс верификации обычно занимает от трех до пяти рабочих дней но может затянуться до двух недель в сложных случаях.

Создайте Ad Account перейдя в Business Settings затем Ad Accounts и выбрав Add для создания нового рекламного аккаунта. Привяжите платежный метод добавив кредитную или дебетовую карту для оплаты рекламы.
Создайте Access Token для программного доступа к API перейдя в Business Settings затем System Users и создав нового пользователя с административными правами. Сгенерируйте Token выбрав необходимые права включая ads_management для управления рекламными кампаниями, pages_manage_posts для публикации постов на странице и catalog_management для управления каталогами товаров. Важно выбрать опцию создания токена который не истекает чтобы избежать необходимости регулярного обновления.

Сохраните Business ID найденный в Business Settings в разделе Business Info, Ad Account ID отображаемый в Ads Manager и Access Token полученный при создании System User в безопасном месте для последующей интеграции.

### AutoDS Dropshipping Automation Platform
**Время выполнения**: пятнадцать минут  
**Стоимость**: Advanced план тридцать долларов в месяц рекомендуется для полного функционала

**Детальный гайд**: [SETUP_AUTODS.md](SETUP_AUTODS.md)

**Шаги выполнения:**

Зарегистрируйтесь на платформе autods.com создав новый аккаунт. Выберите подходящий план где Advanced план за тридцать долларов в месяц включает неограниченное количество товаров, автоматическое размещение заказов и мониторинг цен что критично для автоматизации. Starter план за двадцать долларов дешевле но имеет ограничение на количество товаров.

Подключите торговые каналы начав с eBay US перейдя в Settings затем Selling Channels, выбрав Add eBay и пройдя процесс авторизации где eBay запросит разрешение на доступ для AutoDS. Затем подключите Shopify аналогичным образом выбрав Add Shopify и авторизовав подключение.

Получите API Key перейдя в Settings затем API и нажав Generate Key. Этот ключ позволит n8n workflows взаимодействовать с AutoDS программно для поиска товаров и автоматического выполнения заказов.

Настройте автоматическое размещение заказов перейдя в Settings затем Automation и активировав Enable Auto-ordering. Установите Safety stock level на значение три что означает что система не будет автоматически заказывать товар если у поставщика остаток меньше трех единиц, предотвращая ситуации с товарами которые могут закончиться во время выполнения заказа.

Сохраните полученный API Key так как он потребуется для интеграции с системой агентов.

### n8n Workflow Orchestrator
**Время выполнения**: от тридцати до шестидесяти минут в зависимости от выбранного варианта  
**Стоимость**: Cloud версия двадцать долларов в месяц или Self-hosted от пяти до десяти долларов в месяц за VPS

**Детальный гайд**: [SETUP_N8N.md](SETUP_N8N.md)

**Выбор варианта развертывания:**

Вариант A использует n8n Cloud что проще в настройке и не требует технических знаний. Зарегистрируйтесь на n8n.io выбрав подходящий план где Starter за двадцать долларов в месяц включает пять тысяч выполнений workflows что достаточно для начала. Сохраните URL вашего cloud instance который будет выглядеть как https://ваш-инстанс.app.n8n.cloud и потребуется для настройки webhooks.
Вариант B предполагает Self-hosted развертывание что дешевле в долгосрочной перспективе но требует базовых технических навыков. Арендуйте VPS сервер на одном из популярных провайдеров таких как DigitalOcean, Hetzner или Vultr где базовый план стоит от пяти до десяти долларов в месяц и включает достаточные ресурсы для работы n8n.

Установите n8n через Docker что является самым простым способом развертывания. Подключитесь к серверу через SSH и выполните команду запуска контейнера указав порт пять тысяч шестьсот семьдесят восемь для веб-интерфейса и примонтировав том для сохранения данных workflows. После запуска n8n будет доступен по IP адресу вашего сервера на указанном порту.

Опционально настройте доменное имя такое как n8n.yourdomain.com для удобного доступа и настройки HTTPS сертификата через Let's Encrypt что обеспечит безопасное соединение. Включите Webhook URL в настройках n8n для приема внешних событий от Shopify и других сервисов.

Сохраните Instance URL вашего развернутого n8n который потребуется для настройки интеграций с другими сервисами.

### OpenAI API для AI агентов
**Время выполнения**: десять минут  
**Стоимость**: оплата по факту использования, рекомендуется начать с пополнения на сумму от двадцати до пятидесяти долларов

**Детальный гайд**: [SETUP_OPENAI.md](SETUP_OPENAI.md)

**Шаги выполнения:**

Зарегистрируйтесь на платформе platform.openai.com создав аккаунт через email или аккаунты Google, Microsoft либо Apple. Добавьте платежный метод привязав кредитную карту для оплаты использования API в разделе Billing.

Пополните баланс на сумму от двадцати до пятидесяти долларов что обеспечит достаточный запас для начального тестирования системы агентов. Для справки модель GPT-4-turbo стоит один цент за тысячу входных токенов, что означает что обработка пятидесяти товаров через Product Sourcing Agent обойдется примерно в один-два доллара.

Создайте API Key перейдя в раздел API Keys и нажав Create new secret key. Ключ будет начинаться с префикса sk- и должен храниться в секрете так как предоставляет полный доступ к вашему аккаунту OpenAI.

Установите лимиты расходов для предотвращения неожиданно больших счетов. Перейдите в Usage limits и установите Set monthly budget например на сто долларов в месяц. Система автоматически остановит выполнение запросов при достижении лимита защищая от перерасхода.

Сохраните полученный API Key в безопасном месте так как он потребуется для настройки AI агентов в n8n workflows.
