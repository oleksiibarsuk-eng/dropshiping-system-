# Настройка AutoDS

## Обзор

AutoDS используется как управляющий слой для автоматизации закупок (auto‑ordering), мониторинга цен поставщиков и синхронизации остатков между различными площадками. Он выступает прослойкой между вашей системой и внешними поставщиками, частично снимая с вас необходимость вручную обрабатывать каждый заказ и отслеживать динамику наличия.

Выбор AutoDS обоснован тем, что он уже содержит встроенные механизмы интеграции с eBay и Shopify, а также набор практик, заточенных под дропшиппинг. Это сокращает объём кастомной разработки и позволяет сосредоточиться на более интеллектуальных слоях: отборе товаров, ценообразовании, аналитике и построении AI‑агентов.

## ⚠️ Важные замечания

AutoDS нередко становится источником инцидентов:
- Ошибки авто‑заказов
- Задержки поставщиков
- Несоответствие товара описанию

Поэтому важно заранее продумать:
- Как эти инциденты будут логироваться (таблица errors в Supabase, база Errors & Incidents в Notion)
- Какие SOP будут запускаться при их возникновении

## ⏱️ Оценка трудоёмкости

~15–25 минут при условии подготовленных аккаунтов eBay и Shopify.

## Шаги настройки

### 1. Регистрация

1. Зарегистрируйтесь на [autods.com](https://www.autods.com)
2. Выберите подходящий план:
   - **Advanced план** ($29.90/месяц) — рекомендуется для полного функционала
   - **Starter план** ($20/месяц) — дешевле, но имеет ограничение на количество товаров

### 2. Подключение торговых каналов

#### Подключение eBay US

1. Перейдите в **Settings** → **Selling Channels**
2. Выберите **Add eBay**
3. Пройдите процесс авторизации, где eBay запросит разрешение на доступ для AutoDS

#### Подключение Shopify

1. В том же разделе выберите **Add Shopify**
2. Авторизуйте подключение аналогичным образом

### 3. Получение API Key

1. Перейдите в **Settings** → **API**
2. Нажмите **Generate Key**
3. Этот ключ позволит n8n workflows взаимодействовать с AutoDS программно для:
   - Поиска товаров
   - Автоматического выполнения заказов
   - Мониторинга цен

### 4. Настройка автоматического размещения заказов

1. Перейдите в **Settings** → **Automation**
2. Активируйте **Enable Auto-ordering**
3. Установите **Safety stock level** на значение **3**
   - Это означает, что система не будет автоматически заказывать товар, если у поставщика остаток меньше трёх единиц
   - Это предотвращает ситуации с товарами, которые могут закончиться во время выполнения заказа

### 5. Дополнительные настройки

1. Настройте правила синхронизации цен
2. Настройте правила синхронизации остатков
3. Настройте уведомления о критических событиях

## Сохранение данных для интеграции

Сохраните следующие данные в хранилище секретов (см. [SECRETS_MANAGEMENT.md](SECRETS_MANAGEMENT.md)):

- **Логин** AutoDS
- **API Key** — полученный в разделе Settings → API
- **Краткое описание подключённых маркетплейсов** — какие каналы подключены и их статус

## Интеграция с системой

AutoDS интегрируется с системой через:
- n8n workflows для автоматизации заказов
- Supabase для логирования инцидентов
- Telegram Bot для уведомлений о проблемах
