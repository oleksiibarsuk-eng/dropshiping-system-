# Brief - Система Автоматизації Дропшиппінгу

## Мета проекту

Створити повністю автоматизовану систему дропшиппінгу з AI-агентами для арбітражу товарів між eBay.de та американським ринком (eBay.com, Shopify, Facebook Marketplace).

## Ключові вимоги

### Бізнес-вимоги
1. **Цільова маржа**: Мінімум 25-30% після всіх комісій
2. **Ніша**: Камери та фотообладнання (Sony, Canon, Fuji, Panasonic, Ricoh)
3. **Діапазон цін**: €300-800 (висока маржа на товар)
4. **Прибуток**: €900-2500/місяць на стартовому рівні

### Технічні вимоги
1. **Модульна архітектура**: Незалежні агенти з чіткими зонами відповідальності
2. **Відмовостійкість**: Multi-sourcing, резервування постачальників
3. **Compliance**: Автоматична валідація лістингів перед публікацією
4. **Моніторинг**: Проактивне відстеження репутації та метрик
5. **AI-оптимізація**: Інтеграція Cursor AI для розробки та підтримки

### Обмеження
1. **Ручна робота**: KYC, верифікація акаунтів, прив'язка карт (неавтоматизовано)
2. **Бюджет**: Стартові витрати ~$150-200/міс (платформи + інструменти)
3. **Масштабування**: Починаємо з 20-50 товарів, потім розширення

## Критерії успіху

1. ✅ MVP запущено за 2 тижні
2. ✅ Перший продаж за 1 місяць
3. ✅ Позитивний ROI за 2 місяці
4. ✅ Автоматизація >80% операцій за 3 місяці

## Етапи розвитку

### Фаза 0 (Тиждень 1)
Ручна настройка інфраструктури: акаунти, API ключі, n8n

### Фаза 1 (Тиждень 2-3)
MVP: Product Sourcing + Listing + Ops (3 базові агенти)

### Фаза 2 (Місяць 2)
Безпека: Multi-Sourcing + Compliance + Reputation

### Фаза 3 (Місяць 3+)
Масштабування: Marketing + Expansion агенти
