# Правило выбора языка общения

## Обязательное правило для сообщений пользователю
- Ответы и тексты **ДОЛЖНЫ** быть на **РУССКОМ ЯЗЫКЕ**.

## Использование языка в коде
- Идентификаторы кода и названия API — на английском.
- Цитаты ошибок — как в источнике.

## Коммиты/PR
- Можно вести на русском. Если нужно, делайте двуязычные заголовки: "websocket: исправлен дребезг pong (fix pong jitter)"

## Исключения
- Имена функций/классов/параметров и внешние API — на английском.
- Сообщения об ошибках и коды — как есть.
- При ссылке на английские доки — краткая русская выжимка плюс оригинал при необходимости.
